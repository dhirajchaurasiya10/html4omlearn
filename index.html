The table-layout specifies how the width of table columns is calculated. The possible values are: auto - when column or cell width are not explicitly set, the column width will be in proportion to the amount of content in the cells that make up the column fixed - when column or cell width are not explicitly set, the column width will not be affected by the amount of content in the cells that make up the column. Setting Styles to Links Links can be styled with any CSS property (e.g., color, font-family, background, etc.). In addition, links can be styled differently, depending on what state they are in. The following pseudo selectors are available: a:link - defines the style for normal unvisited links a:visited - defines the style for visited links a:active - a link becomes active once you click on it a:hover - a link is hovered when the mouse is over it The example below creates a link that will change the style when the mouse is moved over it. The HTML:

This link is hovered when we mouse over it

css:
a: hover {
color: red;
}
note: When setting the style for several link states, there are some order rules:
- a:hover MUST come after a:link and a:visited
- a:active MUST come after a:hover
